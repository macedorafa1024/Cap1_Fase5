## Descrição do Projeto

Este projeto foi desenvolvido no contexto da disciplina de Inteligência Artificial
e tem como objetivo aplicar técnicas de Machine Learning para prever o rendimento
de culturas agrícolas com base em variáveis climáticas e ambientais.

A análise foi realizada utilizando um dataset contendo informações sobre:

- Tipo de cultura
- Precipitação
- Umidade específica
- Umidade relativa
- Temperatura
- Rendimento agrícola

---

## Objetivos

- Realizar análise exploratória dos dados
- Identificar padrões de produtividade com clusterização
- Detectar possíveis outliers
- Construir modelos preditivos para estimar o rendimento das culturas

---

## Técnicas Utilizadas

- Python
- Pandas
- Scikit-learn
- K-Means (Clusterização)
- Regressão Linear
- Decision Tree
- Random Forest
- KNN
- Support Vector Regression

---

## Resultados

Foram testados cinco modelos de regressão para prever o rendimento das culturas.

O modelo **Random Forest** apresentou o melhor desempenho,
com coeficiente de determinação (R²) próximo de **0.99**, indicando
alta capacidade de previsão do rendimento agrícola.
